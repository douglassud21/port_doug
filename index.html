Sobre Mim

Olá, meu nome é Douglas, sou apaixonado por tecnologia e desenvolvimento de software. Atualmente, estou cursando **Análise e Desenvolvimento de Sistemas** na Universidade Cruzeiro do Sul, onde estou adquirindo conhecimentos fundamentais sobre programação, arquitetura de sistemas e desenvolvimento de soluções tecnológicas inovadoras.

Além disso, tenho uma formação em **Engenharia de Software**. Durante esse curso, aprendi sobre metodologias de desenvolvimento ágil, análise de requisitos e design de sistemas, o que me permitiu aprimorar minha visão sobre como construir aplicações escaláveis e de alta qualidade.

Minha jornada no desenvolvimento front-end começou com uma **formação em Front-End pela Dio.me**, onde adquiri habilidades em HTML, CSS, JavaScript, React, e ferramentas de design, como Figma e Adobe XD. Essa formação me ajudou a entender as melhores práticas para criar interfaces incríveis e funcionais.

Cursos Importantes

  • **Lógica de Programação** - Certificado em Lógica de Programação para resolver problemas de forma eficiente.
  • **Desenvolvimento Web com React** - Curso focado na criação de interfaces dinâmicas utilizando a biblioteca React.
  • **JavaScript para Web** - Aperfeiçoei minhas habilidades em JavaScript para criar aplicações interativas e responsivas.
  • **CSS Avançado** - Curso para aprimorar o uso de CSS em projetos, incluindo Flexbox, Grid e animações.
  • **Design de Interfaces** - Aprendi a projetar layouts que proporcionam uma experiência de usuário única e intuitiva.

Estou constantemente me atualizando com novas tecnologias e frameworks, sempre em busca de oferecer soluções de alta qualidade para os projetos em que trabalho.
